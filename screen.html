ONBOARDING

The user has the option to signup ,login or skip the onboarding page.

HOME

The home page allows the user to search products, navigate to wishlist,profile and cart.The home page consist of trending deals and featured brands.

SEARCH PRODUCTS

The user can search products by clicking on the search icon in the navigation bar.The user can also search the products by applying filters

PROFILE

The profile page allows the user to edit his profile , check his orders, set or change payment options , raise queries.

WISHLIST

The wishlist page allows the user to add his favourite items to wishlist. The user can also add the items to his cart from wishlist.

CART

The cart allows the user to manage the quantity of the products he wish to buy and purchase them.
